# 🧠 Smart Quotes Aggregator (Умный Агрегатор Цитат)

Динамическое Fullstack веб-приложение на Python, которое собирает цитаты из интернета, сохраняет их в базу данных и предоставляет удобный интерфейс для управления коллекцией и API для интеграции.

## 🚀 Основные возможности
* **Автоматический парсинг**: Сбор цитат с веб-ресурса в один клик прямо из интерфейса.
* **Управление данными (CRUD)**: Возможность вручную добавлять новые цитаты и удалять ненужные.
* **Живая статистика**: Подсчет общего количества записей и уникальных авторов в базе данных в реальном времени.
* **Собственный API**: Эндпоинт `/api/quotes`, отдающий всю коллекцию в чистом формате JSON.
* **Модульная архитектура**: Логика сервера и парсера разделена по разным файлам.

## 🛠️ Стек технологий
* **Backend**: Python, Flask
* **Parser**: Requests, BeautifulSoup4
* **Database**: SQLite3
* **Frontend**: HTML5, CSS3, Шаблонизатор Jinja2
